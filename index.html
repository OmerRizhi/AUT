In this task, you will be asked to think of possible uses to a common object.
In each trail you will be given an abject and write down of as many uses for it as you can.
You will have 2 minutes for each trial.
Press Start when you are ready.

You object is: Category.
Please write down as many ways to use the object as possible.
Press Enter after each one.

Group Your Items


You’ll now see the list of answers you wrote earlier. Group them into clusters according to the themes or connections you had in mind.

How to group items:
• Click the first item in the cluster, then click the last item in the cluster — all items between will be grouped together.
• You can only group items that are adjacent to each other in the list.
• If an item should be in a group by itself, click it twice.
• To ungroup a cluster and regroup it differently, click any item within that group — the grouping will then be reset.

You can make as many groups as you like. There are no right or wrong answers.

When you’re done, click Submit.
